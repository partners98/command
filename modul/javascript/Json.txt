───┤ *JSON JAVASCRIPT* ├───

JSON adalah format untuk menyimpan dan mengangkut data.

JSON sering digunakan ketika data dikirim dari server ke halaman web.

Apa itu JSON?

1. JSON adalah singkatan dari J ava S cript O bject N otation
2. JSON adalah format pertukaran data yang ringan
3. JSON tidak bergantung pada bahasa *
4. JSON "mendeskripsikan diri sendiri" dan mudah dimengerti

*Sintaks JSON berasal dari sintaks notasi objek JavaScript, namun format JSON hanya berupa teks. Kode untuk membaca dan menghasilkan data JSON dapat ditulis dalam bahasa pemrograman apa pun.

Contoh JSON

Sintaks JSON ini mendefinisikan objek karyawan: array yang terdiri dari 3 catatan karyawan (objek):

Contoh JSON
{
"employees":[
  {"firstName":"John", "lastName":"Doe"},
  {"firstName":"Anna", "lastName":"Smith"},
  {"firstName":"Peter", "lastName":"Jones"}
]
}

Format JSON Mengevaluasi ke Objek JavaScript

Format JSON secara sintaksis identik dengan kode untuk membuat objek JavaScript.

Karena kesamaan ini, program JavaScript dapat dengan mudah mengubah data JSON menjadi objek JavaScript asli.

Aturan Sintaks JSON

Data dalam pasangan nama/nilai
Data dipisahkan dengan koma
Kurung kurawal menahan benda
Tanda kurung siku menampung array
