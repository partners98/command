────┤ *ICON CSS* ├────

Ikon dapat dengan mudah ditambahkan ke halaman HTML Anda, dengan menggunakan perpustakaan ikon.

Cara Menambahkan Ikon

Cara paling sederhana untuk menambahkan ikon ke halaman HTML Anda adalah dengan perpustakaan ikon, seperti Font Awesome.

Tambahkan nama kelas ikon yang ditentukan ke elemen HTML sebaris mana pun (seperti <i> atau <span>).

Semua ikon di perpustakaan ikon di bawah, adalah vektor terukur yang dapat disesuaikan dengan CSS (ukuran, warna, bayangan, dll.)

<!DOCTYPE html>
<html>
<head>
<script src="https://kit.fontawesome.com/a076d05399.js" crossorigin="anonymous"></script>
</head>
<body>
<i class="fas fa-cloud"></i>
<i class="fas fa-heart"></i>
<i class="fas fa-car"></i>
<i class="fas fa-file"></i>
<i class="fas fa-bars"></i>
</body>
</html>
