────┤ *SYNTAX CSS* ├────

Aturan CSS terdiri dari pemilih dan blok deklarasi.

Pemilih menunjuk ke elemen HTML yang ingin Anda gaya.

Blok deklarasi berisi satu atau lebih deklarasi yang dipisahkan oleh titik koma.

Setiap deklarasi menyertakan nama properti CSS dan nilainya, dipisahkan oleh titik dua.

Beberapa deklarasi CSS dipisahkan dengan titik koma, dan blok deklarasi diapit oleh kurung kurawal.

Dalam contoh ini semua elemen <p> akan diratakan tengah, dengan warna teks merah:

p {
  color: red;
  text-align: center;
} 

Contoh Dijelaskan

1. padalah pemilih dalam CSS (menunjuk ke elemen HTML yang ingin Anda gaya: <p>).
2. coloradalah properti, dan redmerupakan nilai properti
3. text-alignadalah properti, dan centermerupakan nilai properti
